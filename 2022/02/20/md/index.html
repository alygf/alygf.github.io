windows系统部署hexo博客到GitHub

2022-02-20
字数统计: 734   |   阅读时长≈ 2 分钟

准备：（中文玩家如果实在弄不懂，可以参考CSDN,外语很重要！！！）

*安装nodejs

安装nodejs没什么技巧，附上网址 https://nodejs.org/en/
大家根据自己对应的操作系统选择对应的版本号即可
注：大家在安装时，注意有一个add to path 需要选上，即添加到系统环境。
输入node-v即可检查，如出现版本号则代表安装成功。

*安装Git
附上Git的网址git-scm.com
仅有windows版本的网址gitforwindows.org
当然以上两个网站都慢的起飞（不出意外），所以我给大家推荐一个网址 https://registry.npmmirror.com/binary.html?path=git-for-windows/
当然我选择的是2.31.1的64bit的，大家也可以自行选择。（然后安装无脑下一步就好了，自己创个文件夹安装好吧）

*注册GitHub账号，并创建一个仓库
这一步非常重要，你的仓库名字格式必须为usrname.github.io，你只需要修改的是usrname，
后面usrname.github.io就是你的博客网址了。（咱就是说尽量弄一个帅气的名字，不要让进来的游客第一感觉是博主是傻逼）

*检查
1.win+R,输入cmd
2.输入node -v，出现版本号代表安装成功
3.git version出现版本号即为成功

注：在windows系统中，并不是通过sudo提高权限。移动到“开始”右键就能看到管理员权限。

##开始
在自己的本地磁盘中，新建一个文件夹，取名为Bk，用于接下来的操作：
选中这个文件夹，右键选择 git brish here.也可以出现一个命令出现窗口（我建议大家用这个，而不是用win+r然后cmd，因为win+r执行一些命令会出错，而git brish here不会）

输入 但是国内镜像源的速度很慢，所以这里推荐使用命令行去指向淘宝源下载
npm install -g cnpm –registry=https://registry.npm.taobao.org

cnpm安装成功后

然后输入
cnpm install -g hexo-cli

输入 hexo -v 出现版本号即代表成功

然后在命令窗口输入
1.cd:BK
2.hexo init (我觉得是对BK这个文件夹的初始化)

输入 hexo s执行就会出现localhost:4000这个网址，浏览器查看就好

注：我这里没有教hexo的使用，只是带大家利用hexo和GitHub有了免费的服务器和自己的博客
大家可以去学习下hexo，如何写博客，换主题，这样把博客创建出来才有意义

###结束
1.将自己的BK文件里的东西，上传到自己的仓库里。
然后输入hexo g 就可以了。

最后给大家附上_config.yml文件最下方需要添加的格式：

Deployment

Docs: https://hexo.io/docs/one-command-deployment

deploy:
type: git
repository: https://github.com/alygf/alygf.github.io.git
branch: main

否者hexo d 就会执行不成功
